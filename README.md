# blood-donation-backend

Donate & Save is a full-stack digital platform designed to connect blood donors, recipients, and organizations in a fast, reliable, and transparent ecosystem. It streamlines the entire blood donation process from discovering donors to managing requests, running campaigns, and handling financial contributions.

- To empower communities to save lives by making blood donation and medical support fast, accessible, and organized.
  Core Idea:
  🩸 Blood Donation Network
  💳 Campaign & Financial Support
